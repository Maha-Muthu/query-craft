Web Development with Django Framework

This material provides a detailed guide on web application development using the Django framework. It covers installation of necessary tools, setting up Django projects and applications, and using models, views, and templates within the Django architecture. Learning outcomes include understanding the Django MVT architecture, setting up and managing Django projects, creating and using models, views, and templates, and handling database migrations.

Installation

  • Django needs python 3.x.x and above.
  • We use pip for managing packages, pip documentation.
  • To install pip if you don't already have it, there are two ways:
    • Ensurepip
      • python -m ensurepip -upgrade
    • Using get-pip.py script:
      • Get the script from: pip documentation
      • python get-pip.py
  • In order to manage environments per project, we use pipenv.
    • To install run the following: pip install pipenv

Django Framework

  • Python based web development framework.
  • It is follows the model-view-template architecture.
  • Documentation Link
  • Model: A model corresponds to a table in the database, and instance of the model corresponds of the row of the table, the properties of the model make the columns of the table.
  • View: A view can be thought of as the controller, it is a place which decides from where we fetch the data and which template needs to be rendered.
  • Template: This can be thought of as the html page which will be rendered on the web.
