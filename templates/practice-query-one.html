Periodic Table Schema

The PeriodicTable schema is a structured representation of chemical elements, capturing key properties such as atomic number, name, and symbol, along with atomic mass and subatomic particle counts. It includes details on the element's period, phase (like gas or solid), natural occurrence, and type (e.g., nonmetal or noble gas). Each element is uniquely identified by its atomic number, ensuring precise and organized data management.

List the names of nonmetals that are gases.

SELECT ElementName FROM periodictableschema.PeriodicTable WHERE ElementType = 'Nonmetal' AND Phase = 'gas'

What are the top 5 elements with the highest number of neutrons?

SELECT ElementName, NumberOfNeutrons FROM periodictableschema.PeriodicTable ORDER BY NumberOfNeutrons DESC LIMIT 5

Find the lightest element in each phase.

SELECT Phase, ElementName, AtomicMass AS LightestElementMass FROM periodictableschema.PeriodicTable WHERE (Phase, AtomicMass) IN ( SELECT Phase, MIN(AtomicMass) FROM periodictableschema.PeriodicTable GROUP BY Phase )
