Introduction to PostgreSQL: Installation & Configuration

Learn about the installation and setup of PostgreSQL and its client tool, pgAdmin. A guide through installing a Database Management System (DBMS), logging in, importing sample schemas, and using PostgreSQL's features. Supported and unsupported DBMSs and development tools are outlined, with an emphasis on recommended practices. Including deatils on setting up web development stacks (XAMPP, MAMP, LAMP) and generating dummy data for projects. At the end learners will have a comprehensive understanding of PostgreSQL installation, configuration, and basic operations, enabling them to effectively manage and interact with databases using pgAdmin.

The suggested DBMS for this course are:

  • MySQL
  • MariaDB
  • PostgreSQL

Throughout the lectures and lab will be on PostgresSQL

Other DBMS Available include:

  • ORACLE
  • Microsoft SQL Server
  • IBM DB2

Other types of databases include:

  • Key/value stores
  • NoSQL databases
  • Column-oriented databases
  • Document-oriented databases
  • Graph databases
  • Object-Oriented databases
  • Map/Reduce frameworks
  • Semantic data stores

Databases

A database is a structured collection of data organized for efficient retrieval, storage, and management. It serves as a repository where data is stored in a structured format, allowing for easy querying, updating, and analysis. Databases are fundamental in applications ranging from simple websites to complex enterprise systems.
