Introduction to PostgreSQL: Installation & Configuration

Learn about the installation and setup of PostgreSQL and its client tool, pgAdmin. A guide through installing a Database Management System (DBMS), logging in, importing sample schemas, and using PostgreSQL's features. Supported and unsupported DBMSs and development tools are outlined, with an emphasis on recommended practices. Including details on setting up web development stacks (XAMPP, MAMP, LAMP) and generating dummy data for projects. At the end learners will have a comprehensive understanding of PostgreSQL installation, configuration, and basic operations, enabling them to effectively manage and interact with databases using pgAdmin.

Object-Relational Mapping

Object-Relational Mapping (ORM) frameworks simplify database interactions by allowing developers to work with database records using object-oriented programming instead of raw SQL. They enhance productivity, maintainability, and security by abstracting database operations and providing a consistent API. Popular ORM frameworks include Hibernate (Java), Entity Framework (C#), Django ORM (Python), and Sequelize (Node.js).

MySQL-vs-PostgreSQL Comparison

Feature MySQL PostgreSQL
Data Safety Ensures data safety (ACID) based on an explicit setting Always ensures data safety (ACID) by default
Following Rules Doesn't always follow SQL rules strictly Follows SQL rules very strictly
JSON Data Can handle JSON data, but not as well as PostgreSQL Great at handling JSON data, good for NoSQL tasks
Extensibility MySQL has a more rigid system with limited extensibility features. PostgreSQL is highly extensible, allowing users to add new data types, index types, and functions.

PostgreSQL

  • PostgreSQL
  • It is a powerful, open source object-relational database management system (ORDBMS)
  • Since it is open source, it is not controlled by any single company. Thus, its development relies on the global community of developers and companies
  • We can download, install, use and inspect its source code at no cost.
  • Client tool: pgAdmin

pgAdmin is a comprehensive open-source management tool for PostgreSQL, a powerful relational database system. It provides a graphical interface to manage databases, run SQL queries, and administer database operations. pgAdmin supports a wide range of PostgreSQL features.

Using the link below you can use pgAdmin as well to run queries.

Setting up and using pgAdmin in your local machine or use the pgAdmin Tool from here

  • Download and install PostgreSQL
  • The installation includes pgAdmin
  • Select the latest versions
  • Select your operating system
  • Select the defaults (port, locale)
  • Select admin password

Main Window - Tree Control: Server tree and the objects that reside on a server
